# SAE_java2.01P2
## Objectif
 
Application de bureau permettant de gérer une base de données de boîtes de construction (briques, pièces, figurines) et leurs collections personnelles.
 
## Présentation
 
BRIQU'IUTO est une application JavaFX connectée à une base de données MySQL/MariaDB. Elle propose deux espaces :
 
- **Espace Administrateur** : ajout et suppression de boîtes, de pièces, création de thèmes.
- **Espace Collectionneur** : recherche de boîtes (par numéro, par nom, par thème, par pièce contenue), consultation du détail d'une boîte, statistiques, gestion d'une collection personnelle (ajout/retrait, pièces manquantes, composition de boîtes).
